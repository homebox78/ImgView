# ImgZipView

> 보고 · 편집 · 압축까지 — 가볍고 빠른 이미지 뷰어 & 편집·압축 데스크톱 도구

가볍고 광고 없는 **이미지 뷰어 + 가벼운 편집 + 일괄 압축** 올인원 도구입니다.
왼쪽 **폴더 트리(탐색기)** 로 폴더를 넘나들며, 가운데 **바둑판 썸네일**로 훑어보고,
클릭하면 **우측 미리보기(촬영정보 포함)**, 더블클릭하면 **단독 크게보기**로 봅니다.
자르기·회전·반전·보정·크기변경·포맷변환(PDF 포함)·용량 줄이기·워터마크까지 한 곳에서, 여러 장 **일괄 처리**도 가능합니다.

---

## ✨ 주요 기능

### 보기
- **폴더 트리 탐색기** — 로컬 드라이브 + **네트워크/NAS(SMB·UNC)** 탐색. 폴더 클릭 시 이미지 즉시 로드.
- **하위 폴더 카드** — 이미지와 하위 폴더가 함께 있어도 상단에 폴더 카드로 표시(탐색기식 이동).
- **바둑판/자세히 보기** — 썸네일 격자 ↔ 목록(이름·크기·**DPI**·용량·생성일). **정렬**(이름·날짜·용량·해상도, 오름/내림) 저장됨.
- **빠른 로딩** — `file://` 지연 로딩 + 지연 썸네일 + content-visibility로 **수천 장 폴더도 즉시**. 단독뷰는 인접 이미지 **프리로드**로 끊김 없는 넘김.
- **단독 크게보기** — 좌우 이동, 휠/단축키 줌, 드래그 팬, 회전, **슬라이드쇼**, **전체화면(F11)**.
- **촬영정보(EXIF)** — 카메라·촬영일·설정(f값·셔터·ISO·초점거리)·렌즈·**GPS 위치(+지도)**. 미리보기 패널 및 단독뷰 `ⓘ 정보` 오버레이.
- **다크/라이트 테마** 토글 + **뷰어 배경색**(흰/회/검/체크무늬) 선택, **항상 위에(핀)**.
- **지원 포맷** — JPG · PNG · WebP · GIF · BMP · SVG · ICO · **TIFF** · **HEIC/HEIF** · **PSD**(디코더 내장). *(.fig는 미지원)*

### 편집 (선택 이미지 / 여러 장 일괄)
- **자르기(크롭)** · **회전** · **좌우/상하 반전** · **밝기·대비·채도 보정**
- **크기변경**(비율·해상도·긴 축) · **포맷변환**(JPG·PNG·WebP·BMP·**PDF**)
- **용량 줄이기** — WebP/JPG/원본유지·품질·최대크기·새폴더/덮어쓰기
- **워터마크** — 텍스트 또는 **PNG 로고**, 단일 위치 또는 **패턴(반복·기울기)**
- **파일 관리** — 이름변경(일괄 변수: 번호·날짜·시간), 삭제(휴지통)
- 모든 파괴적 편집은 원본을 같은 폴더 `ImgZipView_원본/`에 **자동 백업**.

### 웹페이지 캡처
- 주소 입력 → **전체 페이지(스크롤 포함)** 또는 보이는 영역 캡처 → 그리드에 바로 추가.
- 결과를 **이미지(PNG) 또는 PDF**로, **캡처 이력**(JSON)에서 다시 불러오기.

### 인쇄 / 선택
- 드래그 박스(마퀴) 선택, Ctrl/Shift 다중 선택, 우클릭 컨텍스트 메뉴, 인쇄(Ctrl+P).

> 모든 처리는 PC 내부(로컬)에서만 이루어집니다. 광고 없음.

## ⌨ 단축키

| 키 / 마우스 | 기능 | 키 / 마우스 | 기능 |
|---|---|---|---|
| `←` `→` · `Space` `Backspace` | 이전/다음 이미지 | `Home` `End` | 처음/마지막 |
| `휠` (단독뷰) | 이전/다음 | `Ctrl+휠` · `+` `-` | 확대/축소 |
| `0` / `1` | 창 맞춤 / 원본 100% | `F11` | 전체화면 |
| `S` | 슬라이드쇼 | `Ctrl+P` | 인쇄 |
| 확대 후 **드래그** | 이미지 이동(팬) | `Del` / `F2` | 삭제 / 이름변경 |
| `Ctrl+.` / `Ctrl+,` | 오른쪽/왼쪽 회전 | `Ctrl+R` `Ctrl+E` `Ctrl+K` | 크기변경 / 포맷변환 / 용량줄이기 |
| 그리드 **드래그** | 박스 선택 | `Ctrl+A` | 전체 선택 |

## 🛠 기술 스택

- **Electron 42** — contextIsolation(메인/프리로드/렌더러 분리)
- **Canvas API** — 자르기·회전·반전·리사이즈·보정·워터마크·포맷 인코딩
- 내장 디코더/라이브러리 — **UTIF**(TIFF), **heic2any**(HEIC), **ag-psd**(PSD), **exifr**(EXIF), **jsPDF**(PDF)
- **electron-builder** — Windows NSIS 인스톨러(한국어, 파일 연결)
- **Pretendard** — UI 글꼴(로컬 내장)
