# ImgView (이미뷰)

> 보기 좋게, 빠르게 · ImageZip 시리즈의 이미지 뷰어

가볍고 빠른 **이미지 뷰어 & 기본 편집 데스크톱 도구**입니다.
폴더의 사진을 썸네일로 훑어보고, 크게 보고, 회전·크기변경·포맷변환까지
한 곳에서 처리합니다. 다크 톤 + 민트 포인트의 ImageZip 시리즈 통일 디자인.

## ✨ 주요 기능
- **빠른 뷰어** — 폴더/파일 열기, 썸네일 목록, 좌우 이동, 휠 줌, 창 맞춤, 전체화면
- **드래그앤드롭** — 이미지·폴더를 끌어다 놓으면 바로 열림
- **기본 편집** — 90°/-90° 회전(원본 자동 백업), 크기변경(비율·해상도·긴 축), 포맷변환(JPG·PNG·WebP·BMP)
- **파일 관리** — 이름변경, 삭제(휴지통)
- **탐색기 연동** — 이미지 더블클릭/우클릭으로 바로 열기

> 모든 처리는 PC 내부(로컬)에서만 이루어집니다.

## ⌨ 단축키

| 키 | 기능 | 키 | 기능 |
|---|---|---|---|
| `←` `→` | 이전/다음 | `Home` `End` | 처음/마지막 |
| `휠` | 확대/축소 | `F11` | 전체화면 |
| `Ctrl+T` / `Ctrl+,` | 오른쪽/왼쪽 회전 | `Ctrl+R` | 크기변경 |
| `Ctrl+E` | 포맷변환 | `F2` | 이름변경 |
| `Del` | 삭제(휴지통) | `Esc` | 닫기 |

## 🖥 실행

```bash
npm install
npm start
```

## 📦 설치 파일(.exe) 빌드

```bash
npm run dist
```

`dist/ImgView-Install-<버전>.exe` 가 생성됩니다 (NSIS 설치 마법사).

## 🛠 기술 스택

- **Electron** — 데스크톱 셸
- **Canvas API** — 회전·리사이즈·포맷 인코딩
- **electron-builder** — Windows NSIS 인스톨러
- **Pretendard** — 본문 글꼴 (로컬 내장)

## 📁 구조

| 파일 | 역할 |
|------|------|
| `main.js` | Electron 메인 (창 생성, 파일 열기/저장 IPC) |
| `preload.js` | 렌더러에 안전한 파일 API 노출 |
| `index.html` | 전체 UI + 뷰어/편집 로직 |
| `license_ko.txt` | 설치 시 표시되는 이용약관 |
